Municipal Water Infrastructure Data Viewer for Naperville, Illinois

PROBLEM STATEMENT

IDEAL

  • Efficient and easy to use application for testing the functionality of Geocortex Essentials for municipal infrastructure data.

REALITY

  • Group Tubular Designs #2014 is not familiar with Geocortex and is responsible for creating a data viewer to interact with municipal water infrastructure data.
  • The client (P.E.C.) currently uses Geocortex Essentials 4 for displaying other data.

CONSEQUENCES

  • Currently no way to display the data and demonstrate the end Collaborative Project deliverable.
  • Data viewer created using software not standard to the Client’s normal stack.

PROPOSAL

  • Create a web map viewer using Naperville, Illinois water infrastructure data from ESRI to test functionality with Geocortex Essentials 4.

Methodology

Using Geocortex Essentials 4 and Geocortex Essentials Workflow, a workflow was based off of a query example on Fleming College’s Collaborative Projects of 2019. This example was configured to retrieve a pipe material query for the water infrastructure of Naperville, Illinois. The workflow retrieves query information from a map service published to the Luna server at Fleming College, as such only computers connected directly to it have access and can load the data. Configuring the workflow was a trial and error process, and to demonstrate the learning of Geocortex Essentials Workflow, all iterations of the workflow were left in the final product. Material Search V2 is the last and most polished version of the query workflow.
